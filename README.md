# Sureify Coffee Pricer ☕️

This repository contains a simple TypeScript coding exercise based on the requirements described in:

**`docs/Coding exercise 2026.pdf`**

The goal of the challenge is to implement a pricing function for an automated coffee vending machine that updates the total price as the user changes their selections (e.g. size, creamer).

---

## Setup

Install dependencies:

```bash
npm install
```

## Run Tests

Run the test suite:

```bash
npm test
```

## Format

Format the codebase:

```bash
npm run format
```
